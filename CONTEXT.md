# 今心 ImXin 產品脈絡

今心是以知心四式為核心的情緒覺察與教練產品。本文件記錄產品語言與邊界，避免把 AI 教練、生活支援與心理治療混在一起。

## Language

**知心四式**:
今心的前台情緒整理方法，包含心照、喚名、安神、動念。
_Avoid_: RULER 五步、Mood Meter、Meta-Moment、How We Feel

**阿念教練**:
今心中的 AI 情緒教練，負責接住情緒、整理脈絡，並提出一個可承受的下一步。
_Avoid_: 心理師、治療師、診斷工具、生活管家

**Agentic 情緒代理**:
阿念的 AI 角色定位，能讀取內部脈絡、判斷下一步、使用工具建立與追蹤小行動，並在使用者回來時調整陪跑方向。
_Avoid_: 普通聊天框、純待辦工具、全自主生活代理、治療代理

**完整 Agentic Action Loop**:
阿念每次互動都經過觀察、判斷、執行、保存、評估與調整的閉環，由 LLM 負責語意判斷，由 deterministic code 負責狀態與副作用。
_Avoid_: 單次工具呼叫、純 prompt 回覆、LLM 直接決定獎勵或資料狀態

**B+ 情緒教練**:
一種介於情緒教練與生活介入之間的產品定位，只在使用者明確授權後追蹤小行動與使用個人脈絡。
_Avoid_: 全自主生活代理、準治療 Agent

**小行動**:
使用者在情緒整理後願意嘗試的一個低風險、短時間、24 小時內可回報的下一步。
_Avoid_: 處方、治療作業、重大生活任務

**小行動回報**:
使用者對自己先前選定小行動的明確回覆，例如有做到、沒做到、做了一半或想換一個。
_Avoid_: 自動判斷完成、隱性監控、成效評分

**小行動紀錄**:
阿念為了追蹤 24 小時內小行動而保存的獨立教練紀錄，具備 active、completed、partial、skipped、expired 等狀態。
_Avoid_: 情緒日誌附註、成就紀錄、治療作業

**最小保存**:
只保存完成追蹤所需的最少內容，不保存完整對話、完整觸發事件或家庭細節。
_Avoid_: 全量保存、生活細節備份、隱性個案紀錄

**回來時回顧**:
使用者再次進入 Coach 或 LINE 對話時，阿念對仍有效的小行動做一次溫和回顧。
_Avoid_: 主動推送催辦、未同意提醒、連續追問

**明確確認**:
使用者以清楚肯定的回覆同意把某個建議設為小行動。
_Avoid_: 沉默同意、根據建議自動建立、把一般建議視為承諾

**Coach 小行動追蹤**:
第一版只在 PWA Coach 內建立、回顧與回報小行動的追蹤能力。
_Avoid_: LINE Bot 小行動狀態機、跨入口強制追蹤

**遊戲化待辦**:
把小行動做成可完成、可累積、可獎勵的待辦系統，包含排行榜、經驗值等級、大量每日任務、金幣與商店，但沒有扣分模式。
_Avoid_: 壓力型打卡、完美主義排名、失敗懲罰

**無扣分模式**:
遊戲化待辦只能給予獎勵、鼓勵或重新選擇，不因逾期、未完成或中斷而扣分。
_Avoid_: 扣分、降級、失敗懲罰、羞辱式提醒

**個人排行榜**:
只比較使用者自己的小行動、任務類型、復盤天數與最佳紀錄，不與其他使用者比較。
_Avoid_: 全站排名、好友排名、父母之間比較、自律排名

**每日任務庫**:
一組預先審核過的低風險小行動，阿念可依內部脈絡挑選，使用者也可新增短自訂任務。
_Avoid_: LLM 無限制自由生成、治療處方、重大生活任務

**金幣商店**:
使用金幣兌換裝飾、表達與個人化外觀的商店，不販售核心支持能力。
_Avoid_: AI 次數、SOS 功能、情緒分析、核心教練能力、隱私或備份能力

**經驗值經濟**:
使用 XP 與金幣獎勵使用者完成、部分完成、回報未完成、確認小行動與完成覺察練習。
_Avoid_: 逾期扣分、未完成歸零、只獎勵完美完成

**折扣點**:
未來可少量折抵訂閱費用的獨立促銷點數，必須與 XP 和金幣分開。
_Avoid_: XP 折抵、金幣折抵、無上限折扣、刷每日任務換現金價值

**免費核心照顧**:
所有使用者都能使用的基礎情緒記錄、基礎 Coach 安撫、SOS 與資料隱私能力。
_Avoid_: 付費解鎖 SOS、付費解鎖基礎情緒支持、付費解鎖資料刪除

**阿念 Pro**:
付費方案，主打個人化行動陪跑、重遊戲化成長系統、更長記憶與週期性成長報告。
_Avoid_: 付費心理治療、付費危機處置、單純聊天訂閱

**7 日推動感**:
使用者在開始使用今心後 7 天內，主觀感覺阿念讓自己在生活裡多前進了一點點。
_Avoid_: 7 天治療成效、每天完美打卡、重大生活改變承諾

**小行動閉環**:
阿念提出或挑選小行動，使用者明確確認，使用者回來回報結果，阿念再依回報調整下一步。
_Avoid_: 只給建議、只做打卡、沒有回報的任務

**身體降載型小行動**:
以降低生理負荷與注意力負荷為目標的第一類小行動，例如短呼吸、喝水、離開現場或寫一句需要。
_Avoid_: 複雜規劃、重大修復、深度分析

**7 日小陪跑**:
新使用者進入 Coach 的主入口，讓阿念在 7 天內陪使用者完成至少一次小行動閉環。
_Avoid_: 挑戰、訓練營、任務衝刺、治療計畫

**陪跑目標**:
使用者在 7 日小陪跑開始時選定的方向，第一版固定為睡前焦慮、親子衝突修復與每日自我照顧。
_Avoid_: 自由發散目標、治療目標、重大人生目標

**三節點陪跑節奏**:
7 日小陪跑第一版節奏，包含 Day 1 啟動、Day 2-3 回顧閉環、Day 7 小結。
_Avoid_: 每日壓迫式任務、每天長報告、硬性連續 7 天打卡

**行動回顧小結**:
7 日小陪跑結束時的一頁非診斷式回顧，整理回來次數、小行動結果、適合類型、模式觀察與下一週保留動作。
_Avoid_: 心理評估報告、改善百分比、焦慮分數、診斷式語句

**免費首輪陪跑**:
所有使用者都可免費完成一次 7 日小陪跑，用來感受阿念是否真的推動生活一點點。
_Avoid_: 首輪付費牆、先收費再證明價值、付費才有基礎閉環

**迷你陪跑**:
當 7 日小陪跑未完成小行動閉環時，阿念提供的 3 天縮小版陪跑。
_Avoid_: 失敗補考、懲罰回合、羞愧式重來

**復盤獎勵**:
使用者回報小行動結果即可獲得的進度，包含完成、做了一半或沒做到但有回來說明。
_Avoid_: 只獎勵完成、未完成懲罰、連勝壓力

**復盤連續**:
使用者連續多日回來回報或復盤小行動的 streak，獨立於情緒記錄 streak。
_Avoid_: 情緒記錄 streak 混算、連勝、每日壓迫

**個人脈絡連結**:
使用者明確授權後，阿念可參考的外部生活脈絡，用於產生更貼近當下現實的建議。
_Avoid_: 全面監控、背景窺探、未授權讀取

**內部脈絡**:
今心自己產生或由使用者在今心內回報的資料，例如情緒紀錄、LINE 互動、Coach 對話與小行動回報。
_Avoid_: 外部行事曆、Gmail、文件、任務管理工具

## Relationships

- **知心四式** 產生一個或零個 **小行動**。
- **阿念教練** 的產品角色是 **Agentic 情緒代理**，不是普通聊天框或單純遊戲化待辦。
- **Agentic 情緒代理** 必須以 **完整 Agentic Action Loop** 實作。
- **Agentic 情緒代理** 以 **7 日推動感** 驗收，而 **遊戲化待辦** 是它推動生活微行動的工具層。
- **阿念教練** 可以透過 **小行動回報** 追蹤 **小行動**，但不能把它變成治療作業。
- **小行動** 應保存為獨立的 **小行動紀錄**，而不是塞進情緒日誌的調節策略欄位。
- **小行動紀錄** 採 **最小保存**，只保存標題、狀態、期限、來源與使用者短回報。
- **小行動紀錄** 只能在使用者 **明確確認** 後建立。
- **小行動** 超過 24 小時未回報即過期，之後只能被溫和帶過，不能追問。
- **阿念教練** 第一版只能透過 **回來時回顧** 跟進小行動，不主動推送提醒。
- **Coach 小行動追蹤** 可以參考 LINE 產生的情緒紀錄，但第一版不從 LINE Bot 建立小行動。
- **遊戲化待辦** 可以接成就或 streak，但必須避免把未完成的小行動塑造成失敗。
- **遊戲化待辦** 的第一版採 **復盤獎勵**，不只獎勵完成。
- **遊戲化待辦** 採 **無扣分模式**，逾期或未完成不扣分、不降級。
- **遊戲化待辦** 的排行榜只能是 **個人排行榜**。
- **遊戲化待辦** 的大量每日任務來自 **每日任務庫**、阿念挑選與使用者短自訂。
- **金幣商店** 只能販售裝飾、表達與個人化外觀，不得鎖住核心支持能力。
- **經驗值經濟** 獎勵完成最多、部分完成次之、回報未完成仍有獎勵，逾期不給獎勵也不扣分。
- **折扣點** 可作為未來訂閱折扣機制，但不能與 XP 或金幣混用。
- **免費核心照顧** 不可被 **阿念 Pro** paywall 鎖住。
- **阿念 Pro** 的主要價值是個人化行動陪跑與遊戲化成長，不是單純 AI 陪聊。
- **7 日推動感** 是打磨第一版產品體驗的 North Star。
- **7 日推動感** 至少需要一次 **小行動閉環** 與使用者主觀覺得有被推動。
- **7 日推動感** 的第一個閉環優先使用 **身體降載型小行動**。
- **7 日小陪跑** 是新使用者第一次進 Coach 的主入口。
- **7 日小陪跑** 的第一版 **陪跑目標** 固定為睡前焦慮少一點、親子衝突後快一點回來、每天做一個照顧自己的小動作。
- **7 日小陪跑** 採 **三節點陪跑節奏**，不是每天強制塞任務。
- **7 日小陪跑** 以 **行動回顧小結** 收束，不產生心理評估報告。
- **7 日小陪跑** 的第一輪是 **免費首輪陪跑**。
- **7 日小陪跑** 未完成小行動閉環時，阿念應提供 **迷你陪跑** 或縮小小行動，不使用失敗語言。
- **復盤連續** 與情緒記錄 streak 分開計算。
- **B+ 情緒教練** 的第一階段只使用 **內部脈絡**。
- **B+ 情緒教練** 未來可以使用 **個人脈絡連結**，但必須由使用者明確授權。
- **個人脈絡連結** 只能輔助建議，不能讓 **阿念教練** 自動替使用者做重大生活決策。

## Example dialogue

> **Dev:** "阿念看到使用者明天行事曆很滿，可以直接建議他取消會議嗎？"
> **Domain expert:** "不行。阿念可以說明天看起來比較滿，邀請他選一個小行動，例如提前準備或安排 3 分鐘暫停，但不能替他判斷要取消哪個現實承諾。"

## Flagged ambiguities

- "Agentic" 容易被誤解成全自主生活代理；目前 resolved 為 **B+ 情緒教練**，不是準治療或全自主代辦。
- 阿念的 Agentic AI 角色 resolved 為 **Agentic 情緒代理**：會讀脈絡、判斷下一步、使用工具追蹤小行動、回來時調整陪跑；**遊戲化待辦** 只是工具層，不是產品本體。
- Agentic runtime resolved 為 **完整 Agentic Action Loop**，不是目前 production `coach` 的單次 function call fallback；LLM 做語意判斷，deterministic code 管狀態、副作用、獎勵、過期與安全守門。
- "連結各種應用程式" resolved 為 **個人脈絡連結**，前提是明確授權、用途限縮與可撤回。
- 第一批個人化資料 resolved 為 **內部脈絡**，先不接外部行事曆、Gmail、文件或任務工具。
- 第一版 **小行動** 追蹤 resolved 為使用者明確 **小行動回報**，不由阿念自動推測完成狀態。
- 第一版 **小行動** 有效期限 resolved 為 24 小時；逾期未回報不再主動追問。
- 第一版 **小行動紀錄** resolved 為獨立資料模型，不混入 `regulating` 或一般情緒日誌內容。
- 第一版 **小行動紀錄** resolved 為 **最小保存**，不保存完整 Coach 對話、完整觸發事件、家庭衝突細節或第三方敏感資訊。
- 第一版小行動提醒 resolved 為 **回來時回顧**，不發 LINE push 或 PWA notification。
- 第一版 **小行動紀錄** 建立條件 resolved 為使用者 **明確確認**；阿念不能因為自己提出建議就自動建立。
- 第一版小行動入口 resolved 為 **Coach 小行動追蹤**，先只放 PWA Coach，不放 LINE Bot。
- 小行動是否接成就 / streak resolved 為要接，產品語言改為 **遊戲化待辦**；但獎勵重點應放在回報與復盤，而不是完美完成。
- 第一版 **遊戲化待辦** 獎勵 resolved 為 **復盤獎勵**：completed、partial、skipped 有回報都給進度；expired 不給進度也不懲罰。
- 第一版小行動 streak resolved 為 **復盤連續**，與情緒記錄 streak 分開計算。
- 第一版 **遊戲化待辦** resolved 為重遊戲化：包含排行榜、經驗值等級、大量每日任務、金幣與商店，但必須採 **無扣分模式**。
- 第一版排行榜 resolved 為 **個人排行榜**，不做公開、好友或跨使用者排行。
- 第一版每日任務來源 resolved 為 **每日任務庫** + 阿念從庫中挑選 + 使用者短自訂；不讓 LLM 無限制自由生成任務。
- 第一版 **金幣商店** resolved 為只賣裝飾、表達與個人化外觀，不賣 AI 次數、SOS、情緒分析、核心教練、隱私或備份能力。
- 第一版 **經驗值經濟** resolved 為 completed 給最多 XP / 金幣、partial 次之、skipped 有回報也給獎勵、expired 不給也不扣。
- 未來訂閱折扣 resolved 為獨立 **折扣點**，不使用 XP 或金幣直接折抵訂閱費。
- 訂閱定位 resolved 為 **阿念 Pro**：個人化行動陪跑 + 重遊戲化成長系統；**免費核心照顧** 保持免費。
- 產品打磨目標 resolved 為 **7 日推動感**：使用者 7 天內覺得阿念真的讓生活前進了一點點，但不承諾治療效果或重大生活改變。
- **7 日推動感** 的第一版驗收 resolved 為至少一次 **小行動閉環**：提議或挑選、明確確認、回來回報、依回報調整下一步。
- 7 日第一個小行動類型 resolved 為 **身體降載型小行動**，避免一開始就推複雜規劃或重大修復。
- Coach 新使用者入口 resolved 為 **7 日小陪跑**，不使用挑戰、訓練營或任務衝刺等高壓命名。
- **7 日小陪跑** 第一版目標 resolved 為三個固定 **陪跑目標**：睡前焦慮、親子衝突修復、每日自我照顧；預設推薦每日自我照顧。
- **7 日小陪跑** 第一版節奏 resolved 為 **三節點陪跑節奏**：Day 1 選目標與第一個身體降載小行動；Day 2-3 回來時回顧並完成閉環；Day 7 產出一頁小結。
- Day 7 小結 resolved 為 **行動回顧小結**：一頁非診斷式行動回顧，不寫改善百分比、焦慮分數、心理評估或診斷語句。
- 付費分層 resolved 為 **免費首輪陪跑**：第一次 7 日小陪跑免費；多輪陪跑、完整任務庫、完整商店、長期回顧與更長記憶可作為 **阿念 Pro** 價值。
- 7 日未完成閉環 resolved 為縮小目標或 **迷你陪跑**，不使用失敗語言、不宣判未完成。
