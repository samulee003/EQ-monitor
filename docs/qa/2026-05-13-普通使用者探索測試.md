# 2026-05-13 普通使用者探索測試

測試目標：扮演第一次對今心有興趣的一般使用者，逐步點擊 PWA 主要功能，記錄會影響理解、信任與發布品質的問題。

測試環境：
- 本機網址：`http://127.0.0.1:5173/`
- 瀏覽器：Google Chrome
- 測試時間：2026-05-13 晚間
- 測試狀態：本機 dev server，已有一筆測試用 情緒紀錄

未執行的高風險操作：
- 未確認刪除任何紀錄，只測到刪除確認視窗並取消。
- 未啟用系統通知權限。
- 未開啟應用鎖，避免觸發重新整理與 PIN 設定。
- 未點擊電話熱線連結。
- 未使用真 LINE 帳號完成實際綁定。

## 已測功能

- 首頁 / 安定室：導覽、快速紀錄、今心主流程、Coach FAB。
- 今心主流程：選狀態區、身體掃描、情緒命名、強度、摘要、歷史保存。
- 快速紀錄：象限、情緒、強度、情境、文字備註、回饋、保存。
- 歷史紀錄：空狀態、有資料狀態、象限 chips、編輯、刪除確認、匯出預覽。
- 洞察：週報洞察、熱點圖、指標卡。
- 成就：進度、已解鎖/未解鎖徽章。
- 設定：主題切換、提醒設定彈窗。
- Coach：歡迎訊息、LINE 綁定錯誤、快速呼吸、SOS 緊急安定練習、聊天送出、底部導覽。

## 發現問題

### P0 - 快速紀錄顯示保存成功，但沒有進入歷史

重現步驟：
1. 首頁點「快速紀錄」。
2. 選「很滿 / 卡住」。
3. 選「焦慮」，強度維持 5。
4. 選情境「3C產品」，輸入備註。
5. 點「完成記錄」後再點「保存記錄」。
6. 到「紀錄」頁查看。

實際結果：歷史頁仍顯示空狀態，沒有新增紀錄。

使用者感受：這會直接破壞信任。使用者已經看到「保存記錄」，但回頭找不到資料。

預期結果：快速紀錄應與 今心主流程一樣寫入歷史，或文案改成「只產生回饋，不保存」。

### P1 - 多處固定導覽列遮住主要內容

重現位置：
- 首頁頂部標題「現在的你，感覺如何？」被上方導覽遮住。
- 今心四步 強度頁與摘要頁捲動後，導覽列蓋在卡片中間。
- 歷史頁標題區捲動後也會被導覽遮住。

使用者感受：畫面美感有提升，但像被一片玻璃壓住，普通使用者會覺得頁面沒有對齊或壞掉。

預期結果：主內容需要足夠 top padding / scroll margin，避免 sticky header 覆蓋內容。

### P1 - Coach 的 LINE 綁定區被聊天輸入列擠壓，普通點擊容易失敗

重現步驟：
1. 進入 Coach。
2. 在 LINE 綁定碼輸入 `ABC123`。
3. 點「綁定」。

實際結果：第一次普通點擊沒有明顯回饋；後續強制點擊才出現「綁定失敗：API 錯誤: Binding code not found or expired」。

使用者感受：LINE 綁定區在聊天輸入列旁邊/後方，看起來像被底部 dock 壓住，容易不知道是否有按到。

預期結果：LINE 綁定卡與聊天輸入 dock 要有清楚間距，錯誤提示要直接可見。

### P1 - 歷史頁象限 chips 看起來可篩選，但點擊沒有作用

重現步驟：
1. 歷史頁已有一筆綠色紀錄。
2. 點「高能低悅 0」chip。

實際結果：畫面沒有變化，chip 狀態也沒有改變。

使用者感受：這看起來像篩選器，但不能篩選，會被理解成壞掉。

預期結果：要嘛實作篩選，要嘛改成非按鈕統計卡。

### P2 - 歷史編輯保存後不即時反映，需要離開再回來才看到

重現步驟：
1. 歷史頁點紀錄右側「編輯」。
2. 輸入表達文字並按「儲存」。
3. 留在同頁查看。

實際結果：畫面仍顯示「這筆紀錄尚未寫下表達內容」；離開歷史頁再回來後才看得到剛才保存的文字。

使用者感受：使用者會以為儲存失敗。

預期結果：按「儲存」後同頁立即顯示更新內容，統計也立即更新。

### P2 - 快速紀錄內容偏親職，不像一般情緒紀錄

重現步驟：
1. 首頁點「快速紀錄」。
2. 選紅色狀態。

實際結果：情緒選項與說明都是親職場景，例如「講了三遍還是不收拾玩具」、「孩子比其他同學矮/發展慢」。

使用者感受：如果普通使用者不是父母，會覺得這個 APP 好像不是給自己用的。

預期結果：快速紀錄應使用一般情緒詞；親職版應只在使用者選擇親職角色時出現。

### P2 - Coach 回覆暴露工具細節，且語氣像在宣稱已保存

重現步驟：
1. Coach 點 SOS。
2. 完成 緊急安定練習，選「喝一杯水」。

實際結果：Coach 回覆包含「已為您執行 `save_ruler_log`」。

使用者感受：普通使用者不應看到內部工具名稱，且「已保存」需要能在可見歷史/週報中對得上。

預期結果：隱藏工具名稱，改成自然語句，例如「我已幫你整理成一段覺察紀錄」。

### P2 - Coach 內部底部導覽與整體語言不一致

重現位置：Coach 底部導覽顯示 `Sanctuary`、`Log`、`Coach`、`Insights`。

使用者感受：前面都是繁中語境，這裡突然變英文，像 prototype 還沒收尾。

預期結果：改成「安定室 / 紀錄 / 教練 / 洞察」。

### P2 - 頂部圓形按鈕只顯示單字，語意不夠清楚

重現位置：全站 header 右側 `勳`、`系`、`訊`。

使用者感受：普通使用者需要猜「勳」是成就、「系」是主題、「訊」是提醒。桌面版空間足夠，單字按鈕不必要。

預期結果：桌面顯示完整文字或圖示加文字；手機才縮短。

### P2 - 設定彈窗文字對比偏低

重現步驟：
1. 點右上角「訊」。

實際結果：提醒設定彈窗在亮色背景與暗遮罩下文字偏灰，看起來像停用狀態。

使用者感受：會以為按鈕不能用，尤其「每日心情提醒」與「應用鎖」說明很淡。

預期結果：提高彈窗背景不透明度與文字對比。

### P3 - 今心四步 進度字母不直覺

重現位置：今心主流程進度顯示 `N / N / L / N / C` 等字母。

使用者感受：一般使用者不懂 N、L、C 代表什麼，也不像 今心四步。

預期結果：改成完整中文步驟或標準 今心四步 字母。

### P3 - 情緒詞有重複

重現位置：很慢 / 順心的情緒命名列表中「平靜的」出現兩次。

使用者感受：小瑕疵，但會讓詞庫看起來不夠精緻。

預期結果：去重或改成更細緻的近義詞。

### P3 - Coach 個人設定按鈕沒有明確功能

重現步驟：
1. 進 Coach。
2. 點右上角「人」。

實際結果：沒有明顯設定頁或提示。

使用者感受：像是未完成按鈕。

預期結果：接到個人設定，或暫時移除/禁用並加說明。

### P3 - 歷史空狀態缺少「開始記錄」主行動

重現位置：沒有任何紀錄時的歷史頁。

實際結果：空狀態只提供「匯入備份檔案」。

使用者感受：新使用者可能不知道下一步要回哪裡開始第一筆。

預期結果：增加「開始第一筆紀錄」按鈕，導回安定室。

## 正向觀察

- 今心主流程能成功建立一筆歷史紀錄。
- 洞察頁與成就頁在有一筆紀錄後會正確更新。
- 匯出預覽可正常顯示 JSON。
- SOS 緊急安定練習 能逐步完成，且熱線資訊可見。
- Coach 一般聊天可送出並收到回覆。
- 主題切換可在 system / dark / light 間循環，最後可回到 system。
- 測試期間未出現前端崩潰或白屏。

## 發布前建議排序

1. 先修快速紀錄不保存、LINE 綁定區被輸入列干擾、sticky header 覆蓋內容。
2. 再修歷史 chips 假按鈕、歷史編輯不即時刷新、Coach 工具名稱外露。
3. 最後修文字一致性、單字 icon、設定對比、詞庫重複等質感問題。

結論：整體已可做封閉內測，但以普通使用者視角，仍不建議今晚直接公開發布。若只做 1-3 人封閉內測，建議至少先修 P0 與 P1。
